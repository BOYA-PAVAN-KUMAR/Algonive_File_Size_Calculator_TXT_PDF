# File Size Calculator (TXT & PDF)

Java Swing application to calculate:
- File size (KB / MB)
- Word count
- Character count
- PDF text extraction using Apache PDFBox

## Features
- Supports .txt and .pdf files
- Clean Swing UI
- Uses Apache PDFBox 2.0.29

## Tech Stack
- Java (Swing)
- Apache PDFBox

## How to Run
1. Clone repository
2. Add JARs from `lib/` to build path
3. Run `FileSizeCalculatorPDF.java`
